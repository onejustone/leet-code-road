# 双向链表

双向链表的优点：
- DLL 可以向前或者向后遍历

在单链列表中，要删除一个节点，需要指向前一个节点的指针。 为了获得该先前节点，有时会遍历列表。
在DLL中，我们可以使用先前的指针获取先前的节点。

双向链表的缺点：
- previous 需要额外的存储空间
- 所有的操作都需要额外的维护一个 previous
