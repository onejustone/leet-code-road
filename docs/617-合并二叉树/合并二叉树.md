# 617-合并二叉树

[617-leet-code](https://leetcode-cn.com/problems/merge-two-binary-trees/)

## 初始工作

我们以二叉搜索树为例，分别构造两颗二叉树如下：

`tree1` 的输入顺序是: ` 4 2 1 3 5 6 7`

```typescript
const tree1 = new BinarySearchTree(4);
tree1.insert(2);
tree1.insert(1);
tree1.insert(3);
tree1.insert(5);
tree1.insert(6);
tree1.insert(7);
```

`tree1` 的结构如下：

```bash
     4
    |  \
    2   5
   | \   \
   1  3   6
           \
            7
```

`tree2` 的输入顺序是: `4 2 1 3 6 5 7`

```typescript
const tree1 = new BinarySearchTree(4);
tree1.insert(2);
tree1.insert(1);
tree1.insert(3);
tree1.insert(6);
tree1.insert(5);
tree1.insert(7);
```

`tree2` 的结构如下：

```bash
     4
    |  \
    2   6
   | \  | \
   1  3 5  7
```

BinarySearchTree 的[代码实现](/code/binary-tree/binary-search-tree.ts)

合并之后的结果将是：

```bash
   8
  |  \
  4   11
 | \  | \
 2 6  5 13
          \
           7
```

关于二叉搜素树请参考[此处](/docs/树/二叉树/二叉搜索树.md)
